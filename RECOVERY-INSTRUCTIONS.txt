# Frever Fitness v5.4

Firebase-backed fitness tracker hosted with GitHub Pages.

## v5.4 changes

- New navy and cool-blue gym theme.
- Panda logo background changed from purple to blue.
- Fixed bottom navigation added.
- Home is always included; choose up to four favourite pages in Settings.
- Workout History statistics are condensed onto one line, with actions below.
- Personal Bests are sorted alphabetically and use a compact row layout.
- PB progress history combines all sets from the same workout date.
- PB graphs use one best-value point per date.

## Uploading to GitHub

Extract the ZIP. Upload the files **inside** the extracted folder directly to the root of the Fitness repository, replacing the existing files.

The repository root should contain:

- `index.html`
- `app.js`
- `styles.css`
- `firestore.rules`
- `README.md`
- `assets/`
- your existing `CNAME`

Do not upload the enclosing version folder itself. Keep the existing `CNAME` file.

No Firebase changes are required.
